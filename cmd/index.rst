Command Line Client
===================

Command List
------------

All commands take the following arguments:

:code:`--api-server`
  The Sandwich Cloud API Server URL | Environment Variable: DELI_API_SERVER

:code:`-d | --debug`
  Turn on debug logging

:code:`--raw`
  See the raw API server output
